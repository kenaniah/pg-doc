# PG::Doc

**This gem is under initial development. 0.1.0 will be the first release.**

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pg-doc'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pg-doc
